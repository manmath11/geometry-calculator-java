# Geometry Calculator (Java)

A simple Java project to calculate properties of basic shapes.

## 🔹 Features
- Circle → Area, Diameter
- Rectangle → Area, Perimeter, Diagonal, Aspect Ratio
- Triangle → Area

## 🔹 Concepts Used
- Java Basics
- Classes & Objects
- Methods
- Math Functions

## 🔹 How to Run
1. Compile:
  GeometryProject.java
2. Run:
   GeometryProject
